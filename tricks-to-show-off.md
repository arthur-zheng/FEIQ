###1. Use ~~ to Get Integer Part
```js
~~1.1     // 1
~~1.8     // 1
~~(-1.1)  // 1
~~(-1.8)  // 1

~~-1.1    // 1

// Works like Java
const three = ~~(10/3);
```

###2. Use !! to Get Boolean
```js
!!5        // true
!!true     // true
!!''       // false
!!null     // false
!![]       // true
```
