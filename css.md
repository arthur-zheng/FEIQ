### Text Space

#####How to display space in HTML text?

Answer: &amp;nbsp;

#####How to display the text `&nbsp;`?

```js
// Text we want to display exactly:
...
The text 'nbsp;' means 'space'
...
```
Answer:
```js
The text 'a&amp;nbsp;' means 'space'    // use &amp; to escape ;
```
